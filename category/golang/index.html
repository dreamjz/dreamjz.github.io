    1. 函数调用

    Go 语言函数调用：

    1. 函数参数按照从右至左的顺序入栈
    2. 函数返回值存储在栈上

    C 和 Go 在函数调用上的差异：

    1. C 使用寄存器和栈传递参数，使用寄存器传递返回值，无法返回多个返回值
    2. Go 使用栈传递参数和返回值，可以返回多个返回值
