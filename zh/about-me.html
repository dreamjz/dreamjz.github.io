    # 2015~2019

    在校期间学习成绩还行，在学习两个专业的课程的情况下，学年成绩均在专业第二。

    课余余时间有折腾过一段时间 unity 和 UE4 的开发，也学过一段时间的 C4D 建模，做一段时间的动画，在 B站投过稿（虽然没啥人气就是，也几年没弄了）

    过了英语六级，自学过日语，尝试去考了 N2 （差一丢丢就及格了/(ㄒoㄒ)/~~）。

    负责编程部分参加过两次数学建模大赛获得过省级二、三等奖

    之后考武汉大学研究生失败，后面需要花时间完成两个专业的论文，等到找工作时已经是春招的尾巴了；

    # 2019~2020

    进了家传统制造业的民企研发中心的Java开发工作，当时也没怎么面试，看着公司规模挺大的，就进去了。

    被派到北京工作，居然住的集体宿舍？？？，比学校的环境还差，当时也不懂太多，硬着头皮先干着~

    期间有在业余时间看书，但无奈环境嘈杂无法集中，遂周末去图书馆占座看书（住在郊区，坐公交也得1个多小时）

    之后疫情爆发，被困在湖北无法返京，却因此被扣去一半月薪？？？人在家中无法出门，之后回到北京又要进行一个多月的隔离。。。心态越来越差，又从同事口中得知只有湖北被无法返京的员工才会被扣工资，交涉无果，遂离职-

    # 2020~2021

    之后来到深圳，开始找工作，已是11月，几次面试碰壁，这才发现自己技术已经大不如前，遂开始复习和学习，但又觉得时间不多了，存款不够支撑多久，越来越焦虑，注意力无法集中~

    之后找了份儿手机银行前后分离项目的后端开发工作，当时觉得挺好的

    2021 年，租了第一个自己的服务器，买了一个域名，开始使用 wordpress 搭建了第一个博客，后来学了 Vue 之后，改成了 vuepress 框架。

    攒了一些钱之后，配了第一台电脑，一次装机点亮，当时特别兴奋~然后自己研究 CPU 和 内存的超频，折腾了有段时间~

    i7-11700K, RTX-3080, 32GB RAM -- Win10 和 Manjaro 双系统

    之后买了个树莓派开发套装（带传感器，LED 灯等配件），不过现阶段只是当做内网服务器来用，部署下 docker 项目之类的~

    带风扇的小家伙就是树莓派了

    在折腾梯子的时候，发现了 crash 这个软件，发现其用 golang 编写，尝试在官网的 playground 上简单学习之后，非常喜欢这个语言，同时也有同学在做这块，觉得 Golang 的前景不错，遂开始学习 Golang
