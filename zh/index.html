    # 目的

    以前遇到好的技术博客，开源书籍或教程，只是浏览一遍或者就直接丢进收藏夹再也没看过了（收藏从未停止，学习从未开始~）

    现在将自己学习的过程记录下来，只有脚踏实地去实践一遍才知道自己理解是否到位，也能增加自己的熟悉程度

    Talk is cheap, show me the code ！

    只停留在理论层面，没有实践，永远不会有进步

    搭建一个网站，一来是为了让自己保持一个高昂的学习热情，二来也可以作为一个知识储备可以随时查看

    将学习内容进行实践和整理在一起，方便自己复习查阅，而不是去茫茫多的收藏夹里面翻找~

    随着网站内容的充实，也让人有点成就感，更好地推动自己去学习~

    还有，首页的背景图是随机的，随到什么图片我也不知道啊 o(≧ 口 ≦)o

    # 内容

    网站内容主要有

    • 学习开源书籍的笔记：会加入自己的理解和示例
    • 学习开源项目的笔记：若开源项目无中文文档，笔记将会以双语的形式记录
    • 自己解决问题的过程：将自己解决问题的过程记录下来，避免遇到同样的问题依然手忙脚乱（多么痛的领悟~）
